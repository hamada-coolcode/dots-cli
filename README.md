
# Welcome

`dots` is a cli that aims to help you:
- manage dotfiles
- manage dependencies of these dotfiles
- download others dotfiles
- switch between dotfiles easily

# Installation

## Building from source

Run:

```bash
git clone https://www.github.com/hamada-ccolcode/dots-cli
cd dots-cli
cargo build
```

## Check if installed

via:

```bash
dots version
```
